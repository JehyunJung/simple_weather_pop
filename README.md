# Simple_Weather_Pop

## Description
Simple Weather notification app

## Development Environment
- Language: JavaScript
- Framework: React-Native(expo based)

## How to use
1. Required Node.js<sup><a href="#fn1">[1]</a></sup>
2. Expo application on your smartphone

```bash
npm install expo-cli
npm install
expo start 
```
